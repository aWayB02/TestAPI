## Технические детали реализации

### Модель Todo
- Поля: id, title, description, completed, created_at, updated_at
- Методы: to_dict(), from_dict(), mark_completed(), mark_uncompleted(), update()

### Репозиторий Todo
- Методы: create(), get_by_id(), get_all(), update(), delete(), filter_by_status()

### Сервис Todo
- Методы: create_todo(), get_todo(), get_all_todos(), update_todo(), delete_todo(), toggle_todo_status()

### API Endpoints
- GET /api/v1/todos - получить все задачи
- GET /api/v1/todos/{id} - получить задачу по ID
- POST /api/v1/todos - создать новую задачу
- PUT /api/v1/todos/{id} - обновить задачу
- DELETE /api/v1/todos/{id} - удалить задачу

## Зависимости проекта

1. Flask - веб-фреймворк
2. SQLAlchemy - ORM для работы с базой данных
3. SQLite - встроенная база данных
4. pytest - фреймворк для тестирования
