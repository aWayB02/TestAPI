# Структура проекта To-Do List

## Общая структура файлов и папок

```
todo-app/
│
├── app.py                 # Точка входа в приложение
├── config.py             # Конфигурационный файл
├── requirements.txt       # Зависимости проекта
├── README.md             # Документация проекта
├── .gitignore            # Игнорируемые файлы для Git
│
├── models/               # Модели данных
│   ├── __init__.py
│   └── todo.py          # Модель задачи
│
├── services/             # Бизнес-логика
│   ├── __init__.py
│   └── todo_service.py  # Сервис для работы с задачами
│
├── repositories/          # Репозитории для работы с БД
│   ├── __init__.py
│   └── todo_repository.py # Репозиторий задач
│
├── api/                  # API endpoints
│   ├── __init__.py
│   └── todo_api.py       # API для задач
│
├── database/            # Работа с базой данных
│   ├── __init__.py
│   └── db.py            # Инициализация БД
│
├── tests/                # Тесты
│   ├── __init__.py
│   ├── test_todo_model.py
│   ├── test_todo_service.py
│   └── test_todo_api.py
│
└── migrations/          # Миграции базы данных (если потребуется)
    └── init_db.py        # Инициализация базы данных
```

## Описание компонентов

### Корневые файлы
- `app.py` - основной файл приложения, запускает сервер
- `config.py` - содержит конфигурационные параметры
- `requirements.txt` - список зависимостей Python
- `README.md` - документация по проекту

### Папка models
Содержит определения моделей данных:
- `todo.py` - модель задачи с полями: id, title, description, completed, created_at

### Папка services
Содержит бизнес-логику приложения:
- `todo_service.py` - сервис для создания, получения, обновления и удаления задач

### Папка repositories
Содержит классы для работы с базой данных:
- `todo_repository.py` - репозиторий для выполнения CRUD операций с задачами

### Папка api
Содержит API endpoints:
- `todo_api.py` - REST API для работы с задачами

### Папка database
Содержит файлы для работы с базой данных:
- `db.py` - инициализация и настройка подключения к SQLite

### Папка tests
Содержит тесты для всех компонентов:
- `test_todo_model.py` - тесты для модели задачи
- `test_todo_service.py` - тесты для сервиса задач
- `test_todo_api.py` - тесты для API задач

### Папка migrations
Содержит скрипты для инициализации и миграции базы данных:
- `init_db.py` - скрипт для создания таблиц в базе данных
